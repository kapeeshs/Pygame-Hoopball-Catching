Game Introduction and controls:

1. Download and Install Python 3.4.0 (64-bit) or above
2. Install Pygame for Python 3.4 (64-bit) and above
3. Read the instructions to play game
4. Right arrow is used for jug to move right and left arrow is used to move left.
5. Up arrow is used to move upward and down arrow is used to move downwards.
6. On holding the buttons jug moves continuously in each direction (upward, downward, right, left).
